# nordelind.skills

Personal Claude Code skills library.

## Install

```bash
claude plugin marketplace add github:lindz0/nordelind.skills
claude plugin install nordelind-skills
```

## Update

```bash
claude plugin update nordelind-skills
```

## Kör en skill från terminalen

Skills kan inte anropas direkt non-interaktivt, men du kan beskriva uppgiften i klartext:

```bash
claude -p "Kör viaplay-sport för kommande helg" --output-format text
```
