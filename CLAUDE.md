## Efter ändringar

Bumpa alltid patch-versionen i `.claude-plugin/plugin.json`, committa och pusha:

1. Öka `version` i `.claude-plugin/plugin.json` (t.ex. 1.0.0 → 1.0.1)
2. `git add -A`
3. `git commit -m "beskrivning av ändringen"`
4. `git push`
